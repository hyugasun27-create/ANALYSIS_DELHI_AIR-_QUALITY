🎯 Project Overview
This project provides a comprehensive analysis of air quality data spanning 3 years (2021-2024) with 1,461 daily records. The analysis focuses on identifying pollution patterns, seasonal trends, and correlations between various pollutants to provide actionable insights for public health and policy-making.

Why This Project Matters
Air pollution is a critical global health crisis, responsible for millions of premature deaths annually. This analysis helps:

🔍 Identify pollution sources and patterns

📊 Quantify health risks across different seasons

🎯 Inform policy decisions with data-driven evidence

📈 Track improvement trends over time

💡 Provide actionable recommendations for stakeholders

📊 Dataset Information
Source
Name: Air Quality Dataset (2021-2024)

Records: 1,461 daily observations

Features: 12+ time and pollutant columns

Period: January 1, 2021 - December 31, 2024
